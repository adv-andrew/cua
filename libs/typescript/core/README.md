# C/ua Core TypeScript Library

The core c/ua library with support for telemetry and other utilities.

## Development

- Install dependencies:

```bash
pnpm install
```

- Run the unit tests:

```bash
pnpm test
```

- Build the library:

```bash
pnpm build
```
